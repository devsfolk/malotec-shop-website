# **App Name**: Malotec Mobile Store

## Core Features:

- Product Catalog: Display a catalog of products, organized by category and subcategory, with filtering.
- Shopping Cart: Allow users to add products to a shopping cart and proceed to checkout.
- Checkout Process: Collect customer information (name, address, delivery note, phone number) during checkout.
- WhatsApp Order Forwarding: Automatically forward order details to the specified WhatsApp number (0344-4850161) upon order placement.
- Admin Dashboard: Provide a password-protected dashboard (accessible via /dmalo) to add, edit, and remove categories, subcategories, and products.
- Real-time Updates: Ensure that changes made in the admin dashboard are immediately reflected in the online store.
- Deep Linking: Implement deep linking for categories, subcategories, and products to allow direct access via URLs, suitable for Facebook ads.

## Style Guidelines:

- Primary color: Strong lime green (#A2F200) to convey freshness and modernity, fitting for a mobile-first store.
- Background color: Very light lime green (#F2F7F0), almost white, to provide a clean and mobile-friendly background.
- Accent color: Cyan (#00FFFF) for highlights, promotions and action items, contrasting with the lime hues.
- Body and headline font: 'PT Sans' for a clean, modern, and readable look on mobile devices.
- Mobile-first design with a two-card-per-row product display, optimized for smaller screens.
- Use simple, clear icons for categories, cart, and other interactive elements to enhance visual cues for users.
- Subtle transitions and animations for loading new products and navigating between pages.
